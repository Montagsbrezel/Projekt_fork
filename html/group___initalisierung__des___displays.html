Detailed Description

The application entry point. Hier ist das Projekt festgehalten. Zusätzlich werden hier die Interrups aufgerufen und das Programm an sich ausgeführt. Hier erfolgt ebenfalls die abfolge mit dem drücken des Buttons, welcher ein Event auslöst und so die Mesung startet.

Return values
int

Initialisiert die Hardwarekomponenten und das Event-System.

Dieser Abschnitt des Codes richtet die LCD-Anzeige, die PWM-Kanäle und die Ereigniswarteschlange ein. Er stellt sicher, dass die Hardware bereit ist und das System in einen definierten Ausgangszustand versetzt wird. Die Ereignisse in der Warteschlange starten anschließend die Anwendung.
