void TestEventLED::handleEvent ( )
overridevirtual

Verarbeitet den aktuellen LED-Zustand und aktualisiert die entsprechenden PWM-Werte.

Diese Funktion passt die RGB-Kanäle einer LED basierend auf dem Zustand p_lLED an. Zusätzlich wird ein Timer für den Blinkmodus gestartet oder gestoppt, falls erforderlich.

  • Setzt die PWM-Werte für die Farbkanäle Rot, Grün und Blau entsprechend des ausgewählten LED-Zustands.
  • Falls der LED-Zustand BLINKING ist, wird ein Timer-Interrupt gestartet, um die Blinkfunktion zu aktivieren.
  • Für andere Zustände wird sichergestellt, dass der Timer deaktiviert ist.
Note
Der LED-Zustand (p_lLED) muss einer der vordefinierten Zustände sein: RED, GREEN, BLUE, BLINKING.
